VENTRASENSE

// POLÍTICA DE PRIVACIDADE

1. Informações Gerais

O VentraSense é um aplicativo de monitoramento meteorológico desenvolvido para uso profissional em aviação e agronegócio. Esta política descreve como coletamos, usamos e protegemos seus dados.

2. Dados Coletados

O aplicativo coleta as seguintes informações para seu funcionamento:

Credenciais de acesso: Nome de usuário e senha para autenticação. As senhas nunca são armazenadas em texto puro.

Dados meteorológicos: Temperatura, umidade, velocidade e direção do vento, precipitação e pressão atmosférica coletados pelas estações cadastradas.

Token de sessão: Um token JWT é armazenado localmente no dispositivo para manter a sessão ativa.

3. Uso dos Dados

Os dados coletados são utilizados exclusivamente para:

— Autenticar o usuário e garantir acesso seguro às estações;

— Exibir dados meteorológicos em tempo real;

— Gerar histórico e alertas configurados pelo próprio usuário.

4. Compartilhamento de Dados

O VentraSense não vende, aluga ou compartilha dados pessoais com terceiros. Os dados meteorológicos são transmitidos exclusivamente entre o dispositivo do usuário e o servidor da estação contratada.

5. Segurança

Toda a comunicação entre o app e o servidor é realizada via HTTPS com autenticação por token JWT. Os dados são armazenados em servidores seguros e não são acessíveis por terceiros.

6. Retenção de Dados

Os dados meteorológicos são retidos por até 30 dias no servidor. O token de sessão é removido ao realizar logout. Dados pessoais são excluídos mediante solicitação.

7. Direitos do Usuário

O usuário pode a qualquer momento solicitar a exclusão de seus dados entrando em contato pelo e-mail abaixo. O app não coleta dados de menores de 18 anos.

8. Contato

Dúvidas sobre esta política podem ser enviadas para: contato@ventrasense.com.br
